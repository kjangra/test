# AlphaLine Techs Website

This is the official website for AlphaLine Techs - a door and gate automation company in Dubai.

## Project Structure

```
alphalinetechs/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # Stylesheet with responsive design
├── js/
│   └── script.js      # JavaScript for interactivity
├── images/
│   └── logo.png       # Logo image (placeholder - replace with actual logo)
└── README.md          # This file
```

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Smooth Scrolling**: Navigation links scroll smoothly to sections
- **Interactive FAQ**: Accordion-style FAQ section
- **Contact Form**: Functional contact form (frontend only)
- **Mobile Menu**: Hamburger menu for mobile devices
- **Modern UI**: Clean, professional design with smooth animations

## Sections Included

1. **Header/Navigation**: Sticky navigation with dropdown menu for services
2. **Hero Section**: Main banner with call-to-action buttons
3. **About Section**: Company information with statistics
4. **Business Associates**: Grid of partner/associate logos
5. **Services Section**: Four main service cards
6. **Testimonials**: Client testimonials section
7. **CTA Section**: Call-to-action section
8. **FAQ Section**: Frequently asked questions with accordion
9. **Contact Section**: Contact information and contact form
10. **Footer**: Quick links and company information

## Setup Instructions

1. Replace the placeholder logo in `images/logo.png` with the actual AlphaLine Techs logo
2. If you have actual images for services, testimonials, or other sections, add them to the `images/` folder
3. Update any placeholder content as needed
4. For production, configure the contact form to send emails to a backend service

## Customization

- **Colors**: Edit CSS variables in `css/style.css` (root section)
- **Content**: Update HTML content in `index.html`
- **Functionality**: Modify JavaScript in `js/script.js`

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- The contact form currently shows an alert on submission. For production use, integrate with a backend service or email API.
- Logo image is a placeholder - replace with actual logo file.
- Font Awesome icons are loaded from CDN for icons.
